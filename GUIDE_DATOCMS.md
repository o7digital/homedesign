## 🛒 Gestión de Productos

### Acceder a los productos:
Menú izquierdo → **Producto** (50 entradas actuales)

### Estructura de un producto:

Usemos el ejemplo de **Triplay** para entender cada campo:

**1. Titulo** (Nombre del producto)
```
Triplay
```
- Nombre comercial del producto
- Debe ser claro y reconocible

**2. Sku** (Código único del producto)
```
HDM67438
```
- Código interno para identificar el producto
- Debe ser único (no repetir)
- Formato sugerido: HDM + números

**3. Categorio Producto** (Categoría)
```
Seleccionar de la lista desplegable
```
- Organiza tus productos por tipo
- Ayuda a los clientes a navegar

**4. imagen** (Imagen del producto)

**Para agregar/cambiar:**
1. Clic en el área de imagen
2. **"Envoyer un nouveau fichier"** → Subir nueva
3. **"Depuis la librairie"** → Elegir existente

**Recomendaciones:**
- Tamaño: 800x600px mínimo
- Formato: JPG (fotos) o PNG (imágenes con fondo transparente)
- Peso: Máximo 1MB
- Calidad: Foto clara, bien iluminada, fondo neutro

**5. Descripcion** (Descripción del producto)
```
Triplay
(Editor de texto con formato)
```

**Consejos para buenas descripciones:**
- Mínimo 50 palabras
- Incluir características técnicas
- Mencionar usos principales
- Destacar ventajas
- Usar lenguaje claro

**Ejemplo de buena descripción:**
```
Triplay de alta calidad, ideal para construcción y mueblería. 
Fabricado con madera de pino seleccionada, ofrece resistencia 
y durabilidad excepcionales. Disponible en diferentes medidas. 
Perfecto para proyectos residenciales y comerciales.
```

**6. Precio** (Precio en MXN)
```
900 MXN
```
- Precio en pesos mexicanos
- Solo números (sin símbolos)
- Ejemplo: 900, 1500, 2350

**7. Disponibilidad** (¿Está disponible?)
```
Toggle: ON (verde) = Disponible
Toggle: OFF (gris) = No disponible
```
- ON: El producto se muestra como disponible
- OFF: El producto aparece como "bajo pedido"

**8. slug** (URL amigable)
```
triplay. xxx
```
- Se genera automáticamente del título
- Usar solo minúsculas y guiones
- Ejemplo: `triplay-premium`, `puerta-madera-cedro`

**9. Fecha Creacion** (Fecha de creación)
```
30/10/2025
```
- Se establece automáticamente
- Opcional: Puedes cambiarla manualmente

### ➕ Agregar un nuevo producto:

**Paso a paso:**

1. **Producto** → Botón **"+ Nouvelle entité"** (verde, arriba a la derecha)

2. **Completar campos obligatorios:**
   - Titulo: Nombre del producto
   - Sku: Código único (ejemplo: HDM00051)
   - Categoria: Seleccionar categoría

3. **Agregar imagen:**
   - Clic en "imagen"
   - Subir foto del producto
   - Asegurarte que sea clara y de buena calidad

4. **Escribir descripción:**
   - Usa el editor de texto
   - Mínimo 50 palabras
   - Sé descriptivo y claro

5. **Establecer precio:**
   - Ingresar precio en MXN
   - Solo números

6. **Activar disponibilidad:**
   - Toggle ON si está disponible
   - Toggle OFF si es bajo pedido

7. **Verificar el slug:**
   - Se genera automáticamente
   - Puedes modificarlo si es necesario

8. **Guardar:**
   - Clic en **"Sauvegarder"**
   - ¡Producto publicado!

### ✏️ Editar un producto existente:

1. **Buscar el producto:**
   - Usar barra de búsqueda en la parte superior
   - O hacer scroll en la lista

2. **Abrir el producto:**
   - Clic en el nombre del producto

3. **Modificar:**
   - Cambiar los campos que necesites
   - Puedes editar uno o varios campos

4. **Guardar:**
   - Clic en **"Sauvegarder"**
   - Los cambios se publican automáticamente

### 🗑️ Eliminar un producto:

⚠️ **Precaución:** Esta acción no se puede deshacer

1. Abrir el producto a eliminar
2. Menú (⋮) arriba a la derecha → **"Supprimer"**
3. Confirmar la eliminación
4. El producto desaparece del sitio

## 💾 Publicación y Guardado

### Estados de un contenido:

#### 🟢 **Publié** (Publicado)
- ✅ Contenido visible en el sitio web
- ✅ Versión actual en línea
- ✅ Clientes pueden verlo

#### 🟡 **Brouillon** (Borrador)
- ⏸️ Contenido en edición
- ⏸️ NO visible en el sitio
- ⏸️ Solo tú puedes verlo

#### 🔵 **In-sync** (Sincronizado)
- 🔄 Versión guardada = versión publicada
- 🔄 No hay cambios pendientes

### Cómo publicar contenido:

**Método automático (recomendado):**
1. Editar tu contenido (texto, imagen, precio, etc.)
2. Hacer clic en **"Sauvegarder"** (botón verde)
3. El estado cambia automáticamente a **"Publié"**
4. Esperar 2-5 minutos
5. Los cambios aparecen en el sitio

**Cómo trabajar en borrador:**
1. Abrir el contenido
2. Menú desplegable (verde) → **"Marquer en tant que brouillon (non-publié)"**
3. Ahora puedes editar sin publicar
4. Cuando estés listo → **"Sauvegarder"** para publicar

### ⏱️ Tiempos de publicación:

| Acción | Tiempo |
|--------|--------|
| Guardar en DatoCMS | Instantáneo |
| Sincronización | 1-2 minutos |
| Aparece en sitio web | 2-5 minutos |
| Total aproximado | **3-7 minutos** |

> **Consejo:** Después de publicar, espera 5 minutos y refresca el sitio con Ctrl+F5 (Windows) o Cmd+Shift+R (Mac)

## 💡 Conseils et Bonnes Pratiques

### 📸 Images:

✅ **Taille recommandée:**
- Slider homepage: 1920x1080px
- Images produits: 800x600px minimum
- Images types de bois: 800x500px

✅ **Format:**
- JPG pour photos
- PNG pour images avec transparence
- Poids max: 2MB par image

✅ **Optimisation:**
- Compresser les images avant upload
- Utiliser des noms descriptifs (ex: `triplay-producto.jpg`)

### ✍️ Textes:

✅ **Description produits:**
- Minimum 50 mots
- Inclure caractéristiques principales
- Utiliser des mots-clés naturellement

✅ **Titres:**
- Courts et descriptifs
- Éviter les majuscules excessives

### 🔍 SEO:

✅ **Slugs:**
- Utiliser des tirets (-)
- Pas d'espaces ni caractères spéciaux
- En minuscules

✅ **Descriptions:**
- Entre 50-160 caractères
- Inclure mots-clés principaux

### 🚫 À éviter:

❌ Ne pas supprimer les contenus par erreur  
❌ Ne pas modifier le schema sans formation  
❌ Ne pas publier de contenu incomplet  
❌ Ne pas utiliser d'images de mauvaise qualité

### 🔄 Workflow recommandé:

1. **Préparer** le contenu (textes + images)
2. **Créer/Modifier** dans DatoCMS
3. **Vérifier** l'aperçu
4. **Sauvegarder** en brouillon
5. **Réviser** le contenu
6. **Publier** la version finale
7. **Vérifier** sur le site en ligne
